# CL_Scan - 獨立式 OCR 文字辨識工具

適用於完全全新的 Windows 電腦，無需安裝任何額外軟體即可使用。

## 📦 打包方法

### 方法一：使用批次檔（推薦）
1. 雙擊 `一鍵打包.bat`
2. 等待打包完成
3. 在 `release/CL_Scan/` 資料夾中找到完整程式

### 方法二：使用 Python 腳本
```bash
python 一鍵打包_完整版.py
```

## ✅ 打包後檢查

執行 `測試打包結果.py` 來驗證打包是否成功：
```bash
python 測試打包結果.py
```

## 📁 打包結果結構

```
release/CL_Scan/
├── CL_Scan.exe              # 主程式（雙擊啟動）
├── CL_Scan.ico              # 程式圖示
├── 使用說明.txt              # 使用者說明
├── 啟動 CL_Scan.bat         # 快速啟動腳本
├── tesseract/               # OCR 引擎（已整合）
│   ├── tesseract.exe        # Tesseract 主程式
│   ├── *.dll                # 所有相依 DLL
│   └── tessdata/
│       └── eng.traineddata  # 英文語言包
└── _internal/               # Python 執行環境
```

## 🚀 使用方法

### 在開發電腦上
1. 執行 `一鍵打包.bat`
2. 等待打包完成
3. 檢查 `release/CL_Scan/` 資料夾

### 在全新電腦上
1. 將整個 `CL_Scan` 資料夾複製到新電腦
2. 雙擊 `CL_Scan.exe` 啟動程式
3. 點擊「截圖辨識」按鈕
4. 拖曳選取要辨識的區域
5. 等待辨識完成，點擊結果即可複製

### 打包參數說明
```python
--onedir          # 打包成資料夾模式（非單一 exe）
--windowed        # 視窗模式（隱藏命令列）
--icon            # 設定程式圖示
--add-data        # 打包額外資料檔案
--hidden-import   # 明確指定要打包的模組
--exclude-module  # 排除不需要的大型模組
--noupx           # 不使用 UPX 壓縮（避免防毒誤判）
```

## 🐛 問題排除

### 打包時遇到錯誤
1. 確認已安裝 Python 3.8+
2. 確認已安裝相依套件：
   ```bash
   pip install -r requirements.txt
   pip install pyinstaller
   ```
3. 確認 `tesseract/` 資料夾存在且包含所有檔案

### 打包後程式無法執行
1. 執行 `測試打包結果.py` 檢查檔案完整性
2. 確認 Windows 防毒軟體沒有阻擋
3. 檢查是否缺少 DLL 檔案

### OCR 無法辨識
1. 確認 `tesseract/tessdata/eng.traineddata` 存在
2. 確認截圖區域文字清晰
3. 查看程式啟動時的 console 輸出（開發模式）
